# house-price-prediction-ml
Machine Learning project to predict house prices using Linear Regression with data preprocessing, feature engineering, and model evaluation.
# 🏠 House Price Prediction using Machine Learning

## 📌 Problem Statement

HomeVista Properties wants to automate house price prediction based on property features such as size, location, and condition.
This project builds a Machine Learning model to predict house prices using historical data.

---

## 🚀 Project Overview

This project uses **Linear Regression** to predict house prices.
The dataset contains features like:

* Lot Area
* Year Built
* Building Type
* Zoning Classification
* Basement Area
* Exterior Type

---

## 🧠 Machine Learning Workflow

1. Data Collection (CSV dataset)
2. Data Preprocessing

   * Handling missing values
   * Encoding categorical variables
3. Feature Selection
4. Model Training (Linear Regression)
5. Model Evaluation

---

## 📊 Model Performance

* **Mean Absolute Error (MAE):** ~34,000
* **R² Score:** ~0.62

👉 The model shows moderate performance and can be improved using advanced techniques.
